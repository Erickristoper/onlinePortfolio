How fast this site went live?

I wanted to have a personal site since college. Well after five years, here it is!

Today, 5th of May 2019, I pushed my initial commit on Github with all courage and did not care. In parallel, I am writing down this essay to document the process real time so here are the things I was able to accomplish:

  • Create a github repo.
  • Pull up W3School website for references on html, css and javascript. Primarily, all things web.
  • Push the very basic-looking website. Not CMS, no back-end. Purely html coded file. But yes! I did not forget to put my face on it.
  • I also started writing this article to be my very first post which I literally copy pasted to the index.html file.

If you were one of the people like me that don’t know where to start, I hope that you can get guidance from it. I intent to organize and create a learning path discussing the development of this site. Hopefully, I’ll get there but for now random as it can be, you’ll see things get added to site.
